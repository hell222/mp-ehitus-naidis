# MP Ehitustööd — disaininäidis

Staatiline disaininäidis (üksik HTML-fail) MP Ehitustööd kodulehe ümberkujundamiseks — premium võtmed-kätte ehitus, terviklik renoveerimine ja restaureerimine Pärnumaal.

**See on ainult disaini näidis ülevaatamiseks**, mitte lõplik leht. `noindex` + `robots.txt` hoiavad selle otsingumootoritest väljas, et see ei segaks mpehitus.ee SEO-d.

- `index.html` — kogu näidis (CSS + JS sees)
- `img/` — pildid ja partnerite logod
